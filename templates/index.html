What is a Smart House?

A smart house is a house that has highly advanced automatic systems for lighting, temperature control, multi-media, security, window and door operations, and many other functions.

What does it include?

  • All house devices are linked.
  • Different sensors in the house provide data about the current conditions which are stored in the cloud.
  • These readings can be easily accessed through a mobile or web application.
  • Remotely adjusting/controlling different devices using your cell phone, tablet or laptop.
  • Receive notifications for some sensor readings specified by the user
